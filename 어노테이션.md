## 어노테이션이란
* 다른 프로그램에게 유용한 정보를 제공하기 위해 사용되는 주석같은 역할을 함
### 역할
* 에러체크,코드 자동 생성,객체간의 관계를 정의함

### 종류
* 표준: 자바가 기본적으로 제공하는 어노테이션
* 메타: 어노테이션을 위한 어노테이션
* 사용자정의 : 사용자가 직접 정의한 어노테이션
  
## 표준 어노테이션
* @Override -> 오버라이딩해라
* @Deprecated -> 앞으로 안씀
* @SuppressWarnings -> 경고를 주지마라
* @FunctionalInterface -> 함수형 인터펲이스이다

## 메타 어노테이션
* @Target -> 큰 클래스에서 많은 어노테이션을 각각 쓸곳을 나눌때
* @Retention -> 유지기간을 지정
    * SOURCE : 소스 파일에만 존재.
    * RUNTIME : 클래스 파일에 존재. 실행시에 사용가능
* @Documented -> 문서 포함을 시킨
* @Inherited -> 어노테이션의 상속
* @Repeatable -> 반복해서 붙일 어노테이션 사용

### Anntation
* 모든 어노테이션의 조상이지만 상속이 불가능함
